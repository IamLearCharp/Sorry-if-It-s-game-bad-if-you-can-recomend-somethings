## 🎮 О игре / About
Консольная RPG игра на C# с выбором героя и врага.  
A console RPG game in C# with hero and enemy selection.

## 🆕 Что нового в 1.1 / What's new in 1.1

- ✅ Выбор цифрами вместо текста / Number selection instead of text
- ✅ При неверном вводе игра переспрашивает / Reprompts on wrong input

## 🚀 Как запустить / How to run
1. Скачай `MyGame.exe` из [Releases](https://github.com/IamLearCharp/Sorry-if-It-s-game-bad-if-you-can-recomend-somethings/releases)
2. Запусти файл
3. Выбери язык: `1` (Русский) или `2` (English)
4. Выбери героя и противника цифрами
